       ```
       - on success: creates notification { type: "clip_synced", title: "Sync Complete", message: `Project "${project.projectName}" synced successfully.`, userId: req.user.id }

       - re-throws error, let outer catch return 500 with { message: err.message }
       - on success: return { success: true, message: "Sync complete" }
       ```

       ```

       - returns { success: true }
       ```

    All routes: try/catch wrapper, 400 on bad input, 500 on unexpected errors.
